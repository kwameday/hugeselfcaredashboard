# Huge Selfcare Performance Dashboard

Dashboard for the period:
**30th October to 30th November 2025**

## Files
- `index.html` – UI
- `styles.css` – styling
- `app.js` – logic + charts + CSV import/export
- `data.json` – your editable data source

## Run locally
Open `index.html` in a browser (or use a simple local server).

## Edit data later
- Preferred: update `data.json` and redeploy
- Also: paste JSON into the UI and click **Update**
- Tables are editable inline; click **Save Table Edits**
  - Stored in browser localStorage (per-device)

## CSV formats
### Categories CSV headers
`product,totalOrders,success,initiated,purchaseInitiated,failure`

### Users CSV headers
`user,attempts`
